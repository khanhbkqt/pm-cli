# Plan 2.1: Wire Template Loaders into milestone.ts and phase.ts

## Objective
Extend `createMilestone` and `addPhase` to accept an optional `projectRoot` parameter. When provided, load and populate the corresponding GSD template and write the result to the filesystem using the helpers from `content.ts`.

This mirrors the pattern already established in `createPlan` (plan.ts) which accepts optional `{ content, projectRoot }`.

## Context
- src/core/milestone.ts
- src/core/phase.ts
- src/core/template_gsd.ts
- src/core/content.ts
- .gsd/templates/milestone.md
- .gsd/templates/phase-summary.md
