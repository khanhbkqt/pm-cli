# Plan 2.1: Tree Rows Progress Indicators

## Objective
Implement progress bars per milestone and phase in the Jira-style hierarchy tree, showing completion percentage based on plan status.

## Context
- .gsd/ROADMAP.md
- dashboard/src/pages/BoardPage.tsx
- dashboard/src/pages/BoardPage.css

## Tasks

<task type="auto">
  <name>Create ProgressBar Component</name>
  <files>
    - dashboard/src/components/ProgressBar.tsx
    - dashboard/src/components/ProgressBar.css
  </files>
  <action>
    Create a new reusable `ProgressBar` React component.
    - It should accept `progress` (0 to 100), `size` ('sm' | 'md' | 'lg', default 'sm') as props.
    - It should use CSS custom properties for colors (`var(--accent-green)` for filled part, color-mix with `var(--text-secondary)` for track).
    - Add smooth CSS transition for the fill width (for animated fills).
    - Ensure it has a container and an inner fill div.
  </action>
  <verify>test -f dashboard/src/components/ProgressBar.tsx && test -f dashboard/src/components/ProgressBar.css</verify>
  <done>ProgressBar component exists and exports correctly.</done>
</task>

<task type="auto">
  <name>Integrate ProgressBar into BoardPage</name>
  <files>
    - dashboard/src/pages/BoardPage.tsx
    - dashboard/src/pages/BoardPage.css
  </files>
  <action>
    Update `MilestoneRow` and `PhaseRow` in `BoardPage.tsx` to include the `ProgressBar`.
    - Calculate `completedPlans` for PhaseRow and `totalCompletedPlans` for MilestoneRow.
    - Compute completion percentage (0 to 100). Guard against division by zero.
    - Import `ProgressBar` and render it inside a new `.tree-row__progress` column.
    - Add `.board-tree__col-progress` to `.board-tree__header` in `BoardPage.tsx`.
    - Update `BoardPage.css` to add flex layout for `.tree-row__progress` and `.board-tree__col-progress` (e.g., width 140px). Hide it on mobile via media queries.
  </action>
  <verify>npm run build --prefix dashboard</verify>
  <done>BoardPage successfully renders progress bars, and the dashboard frontend builds successfully.</done>
</task>

## Success Criteria
- [ ] ProgressBar component is implemented and styled.
- [ ] Completion percentages are accurately calculated for milestones and phases.
- [ ] Progress bars are visible in the hierarchy tree.
